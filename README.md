## 🚀 The Vision

In an era of volatile inflation and shifting global markets, standard budgeting is no longer enough. **Arth-AI** bridges the gap between your daily spending and the macro-economic forces that shape your future. It transforms "raw data" into "economic intelligence."

---

## ✨ Key Features

### 🛡️ Economic Shield Score
Your financial safety net, quantified. Arth-AI analyzes your income-to-essential ratio and spending patterns to calculate your **Resilience Score**.
*   **Tiers:** From *Critical* to *Anti-fragile*.
*   **Stress Testing:** AI-driven analysis of how specific expenses impact your long-term survival.

### ⏳ Inflation Time Machine
Ever wondered what your $5 coffee will cost in 2036?
*   **Historical Context:** Compare current costs to 2016 baselines.
*   **Future Projections:** See the projected cost of your lifestyle 10-20 years from now based on current CPI trends.

### 📈 Spending Velocity
Track the "speed" of your capital outflow.
*   **Multi-Timeframe Analysis:** 7D, 1M, 6M, and 1Y views.
*   **Category Mix:** Deep-dive into where your money is actually going.

### 🧪 What-If Simulator
A sandbox for your future.
*   **Inflation Sliders:** Adjust future inflation rates to see the 10-year cost multiplier.
*   **Cut Scenarios:** Simulate how reducing non-essential spending boosts your Shield Score.

### 🎯 North Star Goals
Don't just save; aim.
*   **Goal Templates:** Quick-start with "Emergency Fund" or "Debt Free" templates.
*   **Priority Focus:** Your "North Star" goal is always front and center.

### 🏆 Gamification & Engagement
*   **Economic Streaks:** Build discipline with daily activity tracking.
*   **Badges & Milestones:** Earn achievements like *Inflation Hedger* and *Macro Expert*.

*   **Frontend:** React 18, Vite, Tailwind CSS (v4), Framer Motion.

*   **Intelligence:** Google Gemini AI (Pro & Flash models) for deep expense analysis and financial advice.
*   **Visualization:** Recharts & D3 for high-density data rendering.

## 📦 Getting Started

### Prerequisites
*   Node.js (v18+)
*   Firebase Project
*   Google Gemini API Key

### Installation
1.  **Clone the repository**
    ```bash
    git clone https://github.com/your-username/arth-ai.git
    cd arth-ai
    ```
2.  **Install dependencies**
    ```bash
    npm install
    ```
3.  **Environment Variables**
    Create a `.env` file in the root:
    ```env
    VITE_FIREBASE_API_KEY=your_key
    VITE_FIREBASE_AUTH_DOMAIN=your_domain
    VITE_FIREBASE_PROJECT_ID=your_id
    GEMINI_API_KEY=your_gemini_key
    ```
4.  **Run Development Server**
    ```bash
    npm run dev
    ```
